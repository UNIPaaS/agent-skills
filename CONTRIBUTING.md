# Contributing

The `unipaas` skill in this repo is generated from the Unipaas developer docs at docs.unipaas.com, so
the fix for a content problem lives in the docs, not here.

## Found a problem with the skill?

- Open an issue in this repo. We triage it here and fix it upstream in the docs.
- Or use the feedback control on any page at https://docs.unipaas.com.

## Pull requests

PRs against the generated paths (`skills/` and the manifest directories) are overwritten on the next
docs sync, so they cannot be merged as-is. For skill content, file an issue and we will fix it at the
source. A PR against the repo's own presentation (the README and the like) is welcome.
